# papersflow-company

This repository contains the PapersFlow Agent Company package in [`papersflow/`](./papersflow).

Install it with `companies.sh`:

```bash
npx companies.sh add papersflow-ai/papersflow-company/papersflow --include company,agents,skills
```

Submit this company to the public directory with:

```text
https://github.com/papersflow-ai/papersflow-company/tree/main/papersflow
```
